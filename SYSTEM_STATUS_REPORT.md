## ⚠️ 已知问题

### 1. Router JSON 解析失败（非 P0 范围）

**问题描述**:
- Router 返回的是 Markdown 格式的文本，而不是 JSON
- 导致 `json.loads()` 失败
- 这是 Router 的 prompt 问题，不是 WebWorker 的问题

**错误日志**:
```
ERROR JSON 解析失败
ERROR 原始内容: 我可以直接回答这个问题，不需要拆解任务。

## 分析结果

- **意图**: information_query（信息查询）
- **置信度**: 1.0
...
```

**影响范围**:
- 仅影响通过 `main.py` 的完整流程
- 不影响 WebWorker 层的功能
- P0 修复的代码都在 WebWorker 层，未受影响

**解决方案**:
1. 修改 Router 的 system prompt，强制返回 JSON
2. 或者在 Router 中添加更强的 JSON 提取逻辑
3. 或者使用结构化输出（如果 LLM 支持）

**优先级**: P1（中等）- 不影响核心功能，但影响用户体验
