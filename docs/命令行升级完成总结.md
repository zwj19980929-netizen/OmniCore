## 🎯 快捷键速查

### 退出
- **Ctrl+C** - 立即退出
- **Ctrl+D** - EOF 退出
- `quit` / `exit` / `q` - 命令退出（仍然支持）

### 导航
- **↑** - 上一条命令
- **↓** - 下一条命令
- **Ctrl+A** - 移动到行首
- **Ctrl+E** - 移动到行尾

### 编辑
- **Ctrl+K** - 删除光标到行尾
- **Ctrl+U** - 删除光标到行首
- **Ctrl+W** - 删除前一个单词

### 其他
- **Tab** - 自动补全（部分支持）
- **Ctrl+L** - 清屏

## 🔧 技术细节

### readline 状态
你的环境已经有 readline 支持，所以所有功能都可以正常使用！


查看历史文件：
```bash
cat ~/.omnicore_history
```
