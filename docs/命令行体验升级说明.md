# OmniCore 命令行体验升级说明

## 已完成的改进

### 1. 优雅退出
- **Ctrl+C**：立即退出，显示"再见！👋"
- **Ctrl+D**：EOF 退出，显示"再见！👋"
- 不再需要手动输入 `exit` 或 `quit`

### 2. 命令历史记录
- **上下方向键**：浏览历史命令
- **自动保存**：历史记录保存到 `~/.omnicore_history`
- **持久化**：重启后历史记录依然存在
- **内置命令**：
  - `history` - 查看最近 20 条命令
  - `clear history` - 清除所有历史记录

### 3. macOS 兼容性优化
- 自动检测 macOS 系统
- 针对 macOS 的 libedit 进行特殊配置
- 提示安装 `gnureadline` 以获得更好体验

### 4. 自动补全（可选）
- **Tab 键**：自动补全常用命令
- 支持中英文命令补全

## macOS 用户建议

为了获得最佳体验，建议安装 `gnureadline`：

```bash
# 安装 readline 库（如果还没有）
brew install readline

# 安装 Python gnureadline 包
pip install gnureadline
```

安装后重启 OmniCore，你将获得：
- ✅ 更流畅的上下键历史浏览
- ✅ 更好的 Tab 补全
- ✅ 更准确的光标移动
- ✅ 支持 Ctrl+A（行首）、Ctrl+E（行尾）等快捷键

## 快捷键列表

### 基础操作
- **Ctrl+C** - 退出程序
- **Ctrl+D** - 退出程序（EOF）
- **↑ / ↓** - 浏览历史命令
- **Tab** - 自动补全（需要 gnureadline）

### 编辑快捷键（需要 gnureadline）
- **Ctrl+A** - 移动到行首
- **Ctrl+E** - 移动到行尾
- **Ctrl+K** - 删除光标到行尾
- **Ctrl+U** - 删除光标到行首
- **Ctrl+W** - 删除前一个单词
- **Ctrl+L** - 清屏（保留当前输入）

### 内置命令
- `history` - 查看历史命令
- `clear history` - 清除历史记录
- `quit` / `exit` / `q` - 退出（仍然支持）

## 技术实现

### 文件修改
1. **main.py** - 集成 EnhancedInput 类

### 核心特性
- 使用 Python `readline` 模块（macOS 使用 `gnureadline`）

## 故障排除

### 问题：上下键不工作
**解决方案**：
```bash
pip install gnureadline
```

### 问题：历史记录不保存
**检查**：
```bash
ls -la ~/.omnicore_history
# 如果文件不存在或无权限，手动创建
touch ~/.omnicore_history
chmod 644 ~/.omnicore_history
```

### 问题：Tab 补全不工作
**原因**：需要 gnureadline 支持
**解决方案**：
```bash
pip install gnureadline
```

### 问题：Ctrl+C 后程序没有退出
**原因**：可能在执行长时间任务
**说明**：Ctrl+C 只在等待输入时立即退出，任务执行中会中断任务
