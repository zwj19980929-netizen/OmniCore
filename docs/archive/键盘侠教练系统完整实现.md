**刀子嘴豆腐心的全局监督者**

## 🚀 工作流程

1. **每个工具执行后**
   - 自动调用 `evaluate_tool_result()`
   - 教练分析结果
   - 显示评论（成功鼓励，失败批评）

2. **失败时的分类**
   - 重复错误 → 严厉批评 + 强制换方法
   - 方向错误 → 引导回正轨
   - 不完整 → 推进完成
   - 超时 → 提速建议
   - 通用失败 → 排查步骤

3. **每次评论都包含**
   - 问题描述（毒舌）
   - 具体建议（引导）
   - 下一步行动（推进）
